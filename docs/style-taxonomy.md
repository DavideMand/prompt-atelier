# Style Taxonomy

Prompt Atelier uses a taxonomy to help users combine style elements deliberately.

## Visual Domains

- Photography: editorial, commercial, documentary, macro, architectural, still life, fashion, food, sports, travel.
- Illustration: editorial, scientific, children's book, technical, botanical, fantasy, sequential, iconographic.
- Graphic Design: poster, packaging, identity system, magazine spread, social ad, album cover, infographic.
- 3D And Product: studio render, CAD-inspired, clay, glass, soft-body, miniature, isometric, exploded view.
- Fine Art Categories: oil painting, watercolor, ink wash, charcoal, fresco, printmaking, collage, textile.
- Interface And Layout: dashboard mockup, mobile screen, landing page hero, app icon grid, presentation slide.

## Style Axes

- Era: ancient, medieval, Victorian, interwar modernism, mid-century, Y2K, contemporary, near-future.
- Finish: matte, glossy, grainy, polished, weathered, handmade, clinical, luxurious, utilitarian.
- Geometry: organic, orthographic, modular, radial, asymmetrical, gridded, cinematic, monumental.
- Density: minimal, sparse, balanced, layered, maximal, archival.
- Color: monochrome, duotone, muted, pastel, jewel-toned, high-key, low-key, complementary.
- Lighting: flat, rim-lit, chiaroscuro, golden hour, fluorescent, studio softbox, neon, moonlit.
- Camera: wide, normal, telephoto, macro, overhead, worm's-eye, isometric, orthographic.
- Material: paper, glass, ceramic, textile, metal, skin, foliage, stone, plastic, liquid.

## Style Safety

Prefer broad public style categories, movements, media, and production language. Avoid asking for exact imitation of living artists or private contemporary creators. When users provide a risky artist-style request, translate it into observable traits such as palette, line quality, composition, medium, and mood.

## Useful Style Recipes

- Museum Product: macro still life, matte plinth, controlled reflections, archival label typography, soft falloff.
- Editorial Tech: clean negative space, precise highlights, modern sans typography, cool neutral field, one vivid accent.
- Cinematic Naturalism: grounded lens language, motivated light, imperfect textures, atmospheric depth.
- Handmade Print: visible ink texture, limited palette, registration offset, paper grain, bold silhouette.
- Luxury Packaging: high contrast, premium materials, restrained typography, sculptural shadows, tactile close-up detail.
